## About The Project

Healthcare Guidance is a front-end healthcare information platform developed to make essential health-related information accessible through a clean, modern, and user-friendly interface.

The website presents information about common medical conditions, treatment approaches, mental health support resources, and emergency care guidance. It is designed with a responsive layout to ensure a seamless experience across desktops, tablets, and mobile devices.

## Features

- Information on common medical conditions
- Detailed treatment pathway section
- Mental health support resources
- Emergency and urgent care guidance
- Responsive design for all screen sizes
- Clean and professional healthcare-focused UI
- Organised and easy-to-read content structure

## Covered Conditions

- Hypertension
- Type 2 Diabetes
- Asthma
- Anxiety Disorders
- Lower Back Pain
- Migraine

## Project Goal

The primary objective of this project is to provide users with a centralised platform for accessing healthcare information in an understandable, organised manner, while promoting health awareness and informed decision-making.

## Disclaimer

This project is intended for educational and informational purposes only. The content provided should not be considered professional medical advice, diagnosis, or treatment. Users should consult qualified healthcare professionals for medical concerns.
